# ADR-007 正式结论必须有证据

- 状态：Accepted
- 日期：2026-07-22

## 背景

系统输出会被投标决策直接引用，无证据的结论（模型幻觉、伪造页码）会造成误导甚至
废标风险。系统只辅助阅读，不代替人工判断（SPEC.md 第 2 节）。

## 决策

正式事实、要求与风险都必须关联有效证据（SourceCitation），可回到文件版本、页码、
原文与坐标。模型不得生成 document_id、page_number 或坐标，这些只能由 clause_id 回查
（SPEC.md 第 7.3 节）；无证据候选不得晋升为正式结果。

## 后果

- 收益：结论可复核、可追溯，降低幻觉风险；
- 代价：抽取与验证链更复杂，覆盖率可能下降，但失败必须可见；
- 无证据不形成正式风险（SPEC.md 第 6.8 节）。

## 重评条件

当证据模型无法支撑某些必要结论类型（如纯推理类提示），重新评估证据定义，但不得
放宽“无证据不晋升”这一安全底线。
