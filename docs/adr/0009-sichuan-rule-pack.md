# ADR-009 四川规则独立扩展包

- 状态：Accepted
- 日期：2026-07-22

## 背景

确定性规则与地区/行业强相关：四川房建与市政施工招标有其特定编号格式、时间规则、
资格与评分惯例。把地区规则写进核心领域会让核心被地区细节污染，也难以扩展其他地区。

## 决策

四川规则放在独立 sichuan_construction 规则包，具有 Manifest、适用范围、版本、依据
来源和兼容 Schema 版本（SPEC.md 第 9.3 节）。核心通过 RulePack 端口加载规则，不
硬编码地区细节。

## 后果

- 收益：核心与地区解耦，未来可新增其他规则包；
- 代价：规则加载需校验适用范围与 Schema 兼容；
- 不适用地区或 Schema 版本的规则包不能加载。

## 重评条件

当规则包数量增多或规则之间出现共享逻辑时，重新评估规则包抽象与共享基类，但保持
“地区规则不进核心”的边界。
