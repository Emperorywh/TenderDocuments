# ADR-006 原文件不可变且 DOCX 生成标准 PDF

- 状态：Accepted
- 日期：2026-07-22

## 背景

证据必须可回到原始文件、版本、页码与坐标。若原文件可被覆盖或 DOCX 页码不稳定，
证据引用会失效，审计与复核失去依据。

## 决策

原始文件与版本只追加、不覆盖；DOCX 经固定 LibreOffice 与字体环境转换为标准化
PDF，所有证据页码基于标准化 PDF（SPEC.md 第 6.4、6.5 节）。对象存储区分 original、
canonical、pages、artifacts、reports、quaternion。

## 后果

- 收益：证据链稳定可复现，版本间可追溯；
- 代价：DOCX 转换需固定镜像层，页码必须统一以标准 PDF 为准；
- 重复哈希文件不创建新有效版本。

## 重评条件

当标准 PDF 转换方案无法保证页码稳定，或需要保留 DOCX 原生结构作为证据时，重新评估
证据页码基准与转换链。
